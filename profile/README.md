# rAthenaFR

rAthenaFR est une initiative francophone autour de rAthena, l'émulateur open-source de serveur Ragnarok Online.

L'objectif du projet est de rendre rAthena plus accessible aux utilisateurs francophones grâce à :

- une documentation française ;
- des guides d'installation Windows et Linux ;
- des explications sur la configuration serveur ;
- des traductions de fichiers utiles ;
- des ressources communautaires pour apprendre à créer et administrer un serveur Ragnarok Online.

## Important

Ce projet est basé sur rAthena et respecte sa licence GPL-3.0.

rAthenaFR n'a pas vocation à remplacer le projet officiel, mais à fournir une base francophone claire, propre et maintenable.
